-----------------------------------------------------------------------------
Requirements for the Coqui TTS API Server
-----------------------------------------------------------------------------
To install these dependencies, run the following command in your terminal:
pip install -r requirements.txt
For GPU support (NVIDIA), first install the PyTorch version with CUDA
by following the instructions at https://pytorch.org/get-started/locally/
Example command for PyTorch with CUDA 12.1 (run this BEFORE pip install -r requirements.txt):
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu121
